## Os 7 riscos e como mitigar

| Risco | Mitigação |
|---|---|
| Conta de anúncios banida | Advertorial como camada intermediária; copy de anúncio dentro das políticas; nada de imitar imprensa. |
| Previsão do El Niño mudar | Revisar o dossiê mensalmente; ter mensagem alternativa pronta ("organize a despensa" funciona mesmo com El Niño fraco). |
| Reembolso alto | Produto que entrega + onboarding bom + copy honesta (medo→alívio, não só medo). |
| Reclamação no Procon/Reclame Aqui | `COMPLIANCE.md` cumprido; garantia honrada rápido; suporte ativo. |
| Gargalo na produção do produto | Começar a Fase 1 hoje; aulas em formato simples (slides narrados). |
| CPA acima do ticket | Não escalar no escuro; testar ângulos; usar order bump/upsell para subir o ticket médio. |
| Janela climática fechando | Lançar rápido; depois de agosto, virar a mensagem para "ainda dá tempo". |
